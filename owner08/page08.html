恆大宣佈恆馳獲2022年 iF 設計大獎，網民：車都沒賣，水份太大了

文章來源：Qooah.com

在4月12日，恆馳官方發布微博，稱旗下的恆馳1 和恆馳6 兩款車型榮獲2022年 iF 設計大獎。該獎項被譽為產品設計界的奧斯卡，和紅點獎、IDEA獎並稱世界三大設計獎。有趣的地方在於，兩款車均為未開始正式發售。在20年8月3日，恆大汽車在上海國際車展展示了恆馳1、2、3、4、5、6、7、8、9九款新車。其中獲獎的恆馳1 為全球首款純電D級旗艦房車，恆馳6 為純電 A級豪華跨界 SUV。對於恆大汽車獲獎，大多數網友們並不認同，有網友表示 iF 獎有很多類別，一般花錢就可以獲得，並沒有實質性的意義。

通過 iF 的官網可以發現，2022年 iF 設計獎的報名費用有不同時間段。按照一件作品為例，早鳥期限（截至2021年6月30日）報名費250歐元；一般報名期限（截至2021年10月15日）報名費350歐元；最後報名期限（截肢至2021年11月19日）報名費450歐元。在早鳥和一般報名期限內報名的話，還能享受買一送一。

入圍後每件作品需要再繳納 200歐元審查費，最終獲獎後還需要繳納 2700歐元用於產品設計包裝費用，或者 1800歐元其他參賽項目服務費，需要外加增值稅費。

按照兩件作品獲獎為例，根據報名時間的不同，其參賽費用在3150歐元至3800歐元之間，大概是人民幣2.19萬元至2.64萬元，這對於企業來說並不算什麼支出。

此次2022年 iF 設計獎共收到 10776 份作品參賽，入圍的產品為 5424件，佔比 50.3%。獲獎的作品有 1973件，金獎作品73件。從數據來看，獲獎的比例大於 10%，機率還是挺高的。

有趣的是，目前獲獎的恆馳1 和恆馳6 兩款車型距離量產還遙遙無期。
