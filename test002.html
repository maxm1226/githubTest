Vivo Pad 是品牌首款平板電腦

除了 Vivo X Fold 和 X Note 這兩款高階手機外，Vivo 今天也如約帶來了自己的第一款平板電腦 Vivo Pad。簡單來說，這是一款主打影音娛樂和輕度辦公的產品，在當下的 Android平板市場中算是非常穩妥的定位。它厚 6.55mm、重 489g，鋁製機身共有藍、灰兩種配色，而且周身沒有天線斷縫。裝置的側邊如 Apple 平板一樣被做平，背後的 13MP 主攝和 8MP超廣角相機則一同出現在和新手機一脈相承的「方圓天階」上。

Vivo Pad 配備了一片 11 吋的 2.5K LCD 面板（16:10 比例），更新率120Hz，支援 DCI-P3 色域、10 億色、HDR10 及杜比視界。螢幕亦支援觸控筆輸入，手指和筆的觸控採樣率分別是120Hz 和 240Hz。原廠的觸控筆擁有鋁合金筆身，支援 4,096 級壓感和側鋒書寫，官方稱若每天使用 2 小時續航力可達到「一整年」。在螢幕的上方設有一顆 8MP 自拍相機，其旁邊還有 ToF模組可用於提醒用戶用眼距離（主要是糾正兒童的習慣）。和今年不少新的 Android 平板一樣，Vivo Pad 也搭載了 Snapdragon 870 處理器，同時有 8GB LPDDR4X RAM 和 128GB /256GB UFS 3.1 儲存。其電池為 8,040mAh，可透過 USB-C 3.1 埠進行最高 44W 有線充電，而且也能輸出最高 4K 60fps 的畫面。除此之外，裝置也內建了 0.7mm振幅的四聲道杜比全景聲喇叭，並能根據螢幕方向自動調節左右立體聲。

不意外的是，Vivo Pad 也能透過磁吸外接鍵盤保護套。官方套背後的支架可調節 0 度至 155 度間的任意角度，64 鍵鍵盤具備 1.3mm 鍵程，下方還設有「同價位最大」的 5,736mm²玻璃觸控板。而在軟體的部分，預載的 OriginOS HD 針對大螢幕改進了 UI 和互動設計。Vivo Pad 和 Vivo 手機間可實現熱點直連、任務接力、畫面分享等功能，並可透過NFC「一碰互傳」當前介面、檔案、圖片等內容。此外螢幕分割、懸浮小窗等常見多工模式也都是有的，Vivo Pad 還能透過「量子套件」跟手機、電腦協作。

如前面所寫，Vivo Pad 分為 128GB 和 256GB 兩個版本，對應首發價分別是人民幣 2,299 和 2,899 元，即日起它便會在中國大陸開啟預購。觸控筆和鍵盤的價格是 349 和 599人民幣，搭配平板一同購買均有 50 元優惠。
